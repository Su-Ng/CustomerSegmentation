library(purrr)
set.seed(123)
# function to calculate total intra-cluster sum of square 
iss <- function(k) {
  kmeans(customer_data[,3:5],k,iter.max=100,nstart=100,algorithm="Lloyd" )$tot.withinss
}

k.values <- 1:10


iss_values <- map_dbl(k.values, iss)

plot(k.values, iss_values,
    type="b", pch = 19, frame = FALSE, 
    xlab="Number of clusters K",
    ylab="Total intra-clusters sum of squares")

K-means Algorithm is used to calculate the clustering algorithm for several values of k. This is done by creating a variation within k from 1 to 10 clusters. Then we compute total intra-cluster sum of square(iss). We then plot iss based on the number of k clusters. The bend or elbow of the plot indicates the optimal number of clusters. In this plot, 4 is the optimal number of clusters,
